# Getting Started

## Packages

```
npm install -g typescript@next
npm install axios
```

## Launch the website

```
npm start
```

go to http://localhost:3000/

## Run Tests

```
npm run test
```

## Deploy

```
npm run build
```

# Other notes

Create a Reactjs template project that includes typescript:

```
npx create-react-app my-react-app-with-ts --template typescript
```

Run package locally:

```
npx serve
```
